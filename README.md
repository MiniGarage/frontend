# Next.js + Tailwind + PWA

This project is a Next.js app router build with Tailwind CSS and `next-pwa`, wired for Privy auth.

**MiniGarage** - Your personal digital garage. Collect legendary racing machines as NFTs on Base blockchain.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

`next-pwa` is disabled in development and will generate the service worker during production builds.
