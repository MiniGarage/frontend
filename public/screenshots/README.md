# Screenshots Directory

Add your app screenshots here for the Base Mini App manifest.

## Requirements:
- **Format**: PNG or JPG
- **Recommended names**:
  - `screenshot1.png`
  - `screenshot2.png`
  - `screenshot3.png`
- **Recommended size**: 1080x1920 (portrait orientation for mobile)
- **Minimum**: 1 screenshot
- **Maximum**: 5 screenshots

These screenshots will be displayed in:
- Base Build preview tool
- App store/discovery pages
- Social media embeds

## Tips:
- Show your app's key features
- Use high-quality images
- Avoid text-heavy screenshots
- Show the app in action
